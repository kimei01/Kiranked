## Kiranked
A simple discord bot to track player statistics and leaderboard data for MSCR RANKED using their API. 
The bot allows users to quickly check MCSR RANKED player stats, rankings, and other information directly from Discord.

## What is MCSR RANKED? 
MCSR RANKED is a mod built for minecraft as a competitive matchmaking system for speedrunners to race against other players in ranked matches and climb a global ladder.

## Features 
- Track player statistics (Rank, Elo, Winrate..) and matches
- Display current season leaderboard of top players and all-time best runs

## Installation 

1. Clone the repository
2. Install dependencies
pip install -r requirements.txt
3. Add your Discord bot token
4. Run the bot

python bot.py
